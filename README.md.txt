#مشروع نظام تسجيل الطلاب (Student Course Registration System)

هاد المشروع عبارة عن نظام لإدارة تسجيل الطلاب بالدورات، اشتغلت عليه بلغة C++ مع تصميم لقاعدة البيانات.

فكرة المشروع: يسمح للمستخدم يدير العمليات الأساسية وهي:
 1- الطلاب: إضافة بيانات طلاب جدد.
 2- الدورات: تعريف دورات تدريبية جديدة.
 3- التسجيل: تسجيل طالب بقلب دورة معينة.
 4- عرض كل القوائم والتسجيلات اللي صارت.

رتبت الملفات بمجلدات مشان يكون الشغل منظم:

مجلد src: هون موجودة كل الأكواد، مقسمة لملفات (.h) و (.cpp) لكل من (الطالب، الدورة، التسجيل) بالإضافة لملف الـ main.

مجلد database: هاد المجلد فيه تصميم قاعدة البيانات اللي اشتغلته على MySQL، رح تلاقي فيه ملفات الـ SQL للجداول والبيانات.

مجلد docs: بيحتوي على ملف الوورد اللي فيه شرح المشروع، ومخطط الـ ERD.
